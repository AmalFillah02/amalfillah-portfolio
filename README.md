### Description

A simple portfolio built with React.

### [live preview](https://amalfillah.vercel.app/)

### Features

- Fully Responsive
- Multi-Page Layout
- Contact Form With EmailJs
- React-Bootstrap
- Edit Content From One Place

### Setup

Get the code

<pre>git clone h</pre>

Install required dependencies

<pre>yarn install</pre>

Start the server

<pre>yarn start</pre>
